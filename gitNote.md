# **Git学习笔记**



- ## **进入git 管理目录，初始化**

  git init



​	查看文件状态

​	git  status 



提交到缓存

git  add 文件名（*所有文件），（.所有未提交文件）

  提交到仓库

git commit -m "备注"



查看提交的版本 

git  log



回退版本

git  reset --hard  "版本号"



查看回退之前的log



git reflog







## 分支

查看分支：

git branch



创建分支

git branch  "分支名字"



切换分支

git checkout "切换到分支的名字"
